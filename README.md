# Dawn Outdoor Products

**Shuguang Outdoor Products** - Premium OEM Manufacturer of Golf Cart Rain Enclosures

## Website Overview

This is a production-ready GitHub Pages website for Dawn Outdoor Products, a leading manufacturer of custom golf cart rain enclosures, weather curtains, and vinyl enclosures.

## Features

✅ **Responsive Design** - Mobile-friendly and works on all devices
✅ **Professional B2B Layout** - Clean, premium American design
✅ **Multiple Pages** - Home, Products, About, Contact
✅ **SEO Optimized** - Meta tags, Open Graph, structured data
✅ **Fast Loading** - Optimized images and assets
✅ **Contact Form** - Email integration for quote requests
✅ **WhatsApp Integration** - Direct messaging buttons
✅ **Gallery Sections** - Product and factory showcases

## Pages

- **index.html** - Homepage with hero banner, features, and CTA
- **products.html** - Detailed product listings and customization options
- **about.html** - Company story, mission, and values
- **contact.html** - Contact information, quote form, and FAQ

## Deployment

This website is designed for GitHub Pages deployment:

1. Push to your GitHub repository
2. Go to Settings → Pages
3. Set source to "Deploy from a branch"
4. Select "main" branch
5. Your site will be live at: `https://yourusername.github.io/dawn-outdoor-products/`

## Image Directories

Please add your product and factory images to the following directories:

- `assets/images/hero/` - Main hero banner image
- `assets/images/products/` - Product images (product-1.jpg, product-2.jpg, product-3.jpg)
- `assets/images/details/` - Detail gallery images (detail-1.jpg through detail-4.jpg)
- `assets/images/factory/` - Manufacturing facility images (facility-1.jpg)
